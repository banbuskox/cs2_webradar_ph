# [CS2 Webradar](https://github.com/clauadv/cs2_webradar) Process Hollowing Program

Credits: [adamhlt](https://github.com/adamhlt/Process-Hollowing)

Usage:
1. Open project in Visual Studio
2. In VS's toolbar hover on `Build` and press `Build Solution`
3. In main folder, new folder will be created named `Release`. Open it and copy file `start_webradar.exe`
4. Paste `start_webradar.exe` into the folder where you have `usermode.exe` file
5. From now on, open `start_webradar.exe` instead of `usermode.exe` everytime you want to start webradar
